# Network Enumeration

＃网络枚举

## Compute

```bash
# List networks
gcloud compute networks list
gcloud compute networks describe <network>

# List subnetworks
gcloud compute networks subnets list
gcloud compute networks subnets get-iam-policy <name> --region <region>
gcloud compute networks subnets describe <name> --region <region>

# List FW rules in networks
gcloud compute firewall-rules list
```

You easily find compute instances with open firewall rules with [https://gitlab.com/gitlab-com/gl-security/security-operations/gl-redteam/gcp\_firewall\_enum](https://gitlab.com/gitlab-com/gl-security/security-operations/gl-redteam/gcp\_firewall\_enum)

您可以轻松地找到具有开放防火墙规则的计算实例，并使用[https://gitlab.com/gitlab-com/gl-com/gl-security/security-poperations/gl-redteam/gcp \ _firewall _enum]（htttps://gitlab.com/ gitlab-com/gl-security/security-operations/gl-redteam/gcp \ _firewall \ _enum）
